# 🚀 AI Sales Platform

An AI-powered Sales CRM built using **React, TypeScript, Tailwind CSS, and Vite**.

The platform helps businesses manage leads, research companies with AI, generate sales emails, and visualize reports through a modern dashboard.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/aman4123/ai-sales-platform.git
```

Go into the project

```bash
cd ai-sales-platform
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```
